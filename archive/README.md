# Archive

This directory contains historical and exploratory materials that are intentionally separated from the primary submission path.

- `archive/records/track_10min_16mb/` and `archive/records/track_non_record_16mb/`: prior run records.
- `archive/root_material/`: prior docs and auxiliary experiment notes.
- `archive/records/legacy_best_run_notes/`: legacy notes/snapshots retained for provenance.

Archived content is retained for traceability but is not the active submission candidate path.
